# Arduino自主學習整理(重製版)

## 簡述

自主學習計畫，我買零件並重製這個專案，並且附帶溫溼度偵測功能；<br>
因為光度偵測準確性不足，為了讓其在臨界點不會太high(爆閃)，我拔掉光敏電阻；設定其定時調整背光，讓我晚上睡覺不受影響。<br>
為了讓房間在電腦關機時仍有時間觀念，並且不用彎下腰看主機的側透，因此我把小型時間顯示裝置移至桌面(指的是物理上的桌子)。

## 材料
- Arduino UNO ($240)
- DS3231 ($50)
- AHT10 ($40)
- LCD2004 I2C ($140)
- 電阻220Rx1 ($1)
- 杜邦線數條
- PN2222A NPN二極體x1
- 1N4007整流二極體x1
- cat.5e網路線50cm

## 接腳定義
| Arduino Pin | AHT10/DS3231/LCD2004 |
| :--: | -- |
| **D4** | SDA |
| **D5** | SCL |

| Arduino Pin | DEF |
| :--: | -- |
| **2** | INPUT_PULLUP |
