# Arduino自主學習整理(重製版)

## 簡述

自主學習計畫，為了把小型時間顯示裝置移至桌面，並且能有溫溼度偵測；<br>
並讓其定時調整背光，讓我晚上睡覺不受影響。<br>
為了讓房間在電腦關機時仍有時間觀念，因此我買零件病蟲致這個專案。

## 材料
- Arduino UNO ($240)
- DS3231 ($50)
- AHT10 ($40)
- LCD2004 I2C ($140)
- 電阻220Rx1 ($1)
- 杜邦線數條
- PN2222A NPN二極體x1
- 1N4007整流二極體x1
- cat.5e網路線50cm

## 接腳定義
| Arduino Pin | AHT10/DS3231/LCD2004 |
| :--: | -- |
| **D4** | SDA |
| **D5** | SCL |

| Arduino Pin | DEF |
| :--: | -- |
| **2** | INPUT_PULLUP |
